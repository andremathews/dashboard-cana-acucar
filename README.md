# 🌾 Dashboard de Produtividade de Cana-de-Açúcar

Dashboard interativo para análise de produtividade e qualidade (ATR) por talhão.

## 🎯 Objetivo
Auxiliar produtores e gestores na tomada de decisão com base em:
- Produtividade média por talhão (t/ha)
- Qualidade da cana (ATR simulado)
- Comparativo entre variedades (RB867515, CTC9, SP80-3280)
- Impacto da chuva na produtividade

## 📁 Arquivos
| Arquivo | Descrição |
|---------|-----------|
| `index.html` | Versão web interativa do dashboard |

## 🛠️ Tecnologias
- Excel (tratamento de dados)
- Power BI Desktop (visualização)
- HTML/CSS/JavaScript + Chart.js (versão web)

## 🚀 Como usar
Abra o arquivo `index.html` em qualquer navegador (Chrome, Edge, etc.)
